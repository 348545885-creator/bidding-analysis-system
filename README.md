# 投标决策辅助分析

基于四川省公共资源交易网实时数据的投标报价辅助分析平台。

**核心流程**：通过网站 ES 搜索 API 获取中标公告列表 → Selenium 自动提取详情页价格 → 基于历史折扣率算法推荐最优报价区间。

## 技术栈

| 层 | 技术 |
|---|------|
| **前端** | React 18 + Babel Standalone + ECharts 5 + Axios（本地静态文件，无 CDN 依赖） |
| **后端** | Python 3 + FastAPI + Uvicorn |
| **爬虫** | API（列表获取）+ Selenium 4 + Edge（详情页价格提取） |
| **数据库** | SQLite |
| **AI**（可选） | 阿里云百炼 qwen-turbo |

## 核心功能

- **实时数据采集**：API 获取列表 + Selenium 提取详情页价格，支持自定义日期范围
- **智能报价分析**：基于相似项目的历史折扣率算法推荐最优报价区间
- **数据可视化**：ECharts 展示 21 个地市州分布和 3 种项目类型分布
- **AI 对话**（可选）：集成阿里云百炼，支持智能问答

## 快速开始

```bash
pip install -r requirements.txt
python app.py
# 浏览器打开 http://localhost:5001
```

双击 `启动服务.bat` 也可一键启动。

### AI 对话配置（可选）

在项目根目录创建 `.env` 文件：

```
ALIBABA_CLOUD_API_KEY=your_api_key_here
```

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/test` | GET | 健康检查 |
| `/api/projects` | GET | 项目列表（分页/筛选） |
| `/api/statistics` | GET | 统计数据 |
| `/api/filter/options` | GET | 筛选选项 |
| `/api/analyze` | POST | 报价区间分析 |
| `/api/trigger-scrape` | POST | 触发数据采集 |
| `/api/ai/chat` | POST | AI 对话 |

内置 Swagger 文档：http://localhost:5001/docs

## 采集规则

- **信息来源**：四川省公共资源交易信息网（ggzyjy.sc.gov.cn）
- **公告类型**：中标候选人公示 + 中标结果公示
- **项目类型**：建筑工程、市政公用工程、水利水电工程
- **列表获取**：ES 搜索 API（`/inteligentsearch/rest/esinteligentsearch/getFullTextDataNew`），支持精确日期范围
- **详情提取**：Selenium Edge 无头浏览器访问详情页，正则提取控制价和成交价
